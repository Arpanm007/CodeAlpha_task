# 🎮 Hangman Game (Python)

A simple **Hangman** game implemented in Python, where you guess a hidden word one letter at a time. You have **6 chances** to guess wrong before the game ends.

---

## 📌 Features
- **Random word selection** from a predefined list.
- Shows **progress** of the guessed word in real-time.
- Keeps track of **correct ✅ and wrong ❌ guesses**.
- Prevents repeated guesses.
- Input validation (only single alphabetic letters allowed).
- Fun console-based game experience.

---

## 🛠️ Requirements
- Python 3.x  
*(No additional libraries required — uses only Python’s built-in modules.)*

---

## 🚀 How to Run
1. **Clone this repository**  
   ```bash
   git clone https://github.com/your-username/hangman-game.git
   cd hangman-game
````

2. **Run the Python file**

   ```bash
   python hangman.py
   ```

---

## 🎯 How to Play

1. The game will **randomly choose a word**.
2. You will see underscores `_` representing each letter.
3. Enter **one letter at a time**.
4. If correct ✅ — the letter appears in the word.
   If wrong ❌ — your remaining chances decrease.
5. Win by guessing all letters before your 6 chances run out.
